Основні концепції

  • PinchButton — основа всіх інтерактивних кнопок
  • ToggleButton — перемикач стану on/off
  • Slider — повзунок для змінних значень
  • @input — зв'язок скрипта зі сценою
  • SceneObject — будь-який об'єкт на сцені

Система компонентів

  • Component — базовий клас (Transform, Script, RenderMeshVisual...)
  • ObjectPrefab — шаблон для клонування
  • instantiate() — створити копію prefab'а
  • getComponent() — отримати компонент об'єкта

1Кнопка + Logger

Найпростіша інтеракція — кнопка виводить текст у консоль

Легко

Що вивчаємо

  • Тип PinchButton — інтерактивна кнопка для Spectacles
  • Callback onButtonPinched.add(callback)
  • Функція print() для виводу в консоль

Код

// @input Component.PinchButton myButton

script.myButton.onButtonPinched.add(function() {
  print("Кнопка натиснута!");
});

Як працює

  1. @input створює зв'язок зі сценою
  2. Користувач робить pinch жест над кнопкою
  3. Викликається функція-callback
  4. print() виводить текст у Lens Studio Logger

2Toggle об'єкта

Кнопка показує/ховає об'єкт на сцені

Легко

Що нового

  • Тип @input SceneObject
  • Властивість enabled — видимість об'єкта
  • Логіка toggle (вкл/викл)

Код

// @input Component.PinchButton toggleButton
// @input SceneObject targetObject

script.toggleButton.onButtonPinched.add(function() {
  script.targetObject.enabled = !script.targetObject.enabled;
  print("Об'єкт: " + script.targetObject.enabled);
});

3Спавн об'єктів

Кнопка створює копії prefab'а і розставляє їх у сітку

Легко

Що нового

  • ObjectPrefab — шаблон для клонування
  • instantiate(parent) — створити копію
  • setLocalPosition() і setLocalScale()

Код

// @input Component.PinchButton spawnButton
// @input Asset.ObjectPrefab cubePrefab
// @input SceneObject container

script.spawnButton.onButtonPinched.add(function() {
  for (var i = 0; i < 5; i++) {
    var obj = script.cubePrefab.instantiate(script.container);
    obj.getTransform().setLocalPosition(new vec3(i * 20, 0, 0));
  }
  print("Створено 5 кубів");
});

4Toggle + Slider

Перемикач показує меню зі слайдером масштабу

Середньо

Що нового

  • ToggleButton — перемикач on/off
  • Slider — повзунок (0..1)
  • onValueUpdate — callback при зміні значення

Код: Toggle

// @input Component.ToggleButton menuToggle
// @input SceneObject menuFrame

script.menuToggle.onButtonPinched.add(function() {
  script.menuFrame.enabled = !script.menuFrame.enabled;
});

Код: Slider

// @input Component.Slider scaleSlider
// @input SceneObject target

script.scaleSlider.onValueUpdate.add(function(value) {
  var scale = 0.5 + value * 2; // 0.5..2.5
  script.target.getTransform().setLocalScale(
    new vec3(scale, scale, scale)
  );
});

5Меню з кнопками

Frame з трьома кнопками для зміни кольору об'єкта

Середньо

Що нового

  • Frame — контейнер UI елементів
  • Робота з кількома кнопками
  • Зміна кольору матеріалу через mainPass.baseColor

Структура сцени

Frame (Container)
├─ RedButton (PinchButton)
├─ GreenButton (PinchButton)
└─ BlueButton (PinchButton)

TargetCube (має матеріал Material)

Код

// @input Component.PinchButton redBtn
// @input Component.PinchButton greenBtn
// @input Component.PinchButton blueBtn
// @input Asset.Material targetMaterial

script.redBtn.onButtonPinched.add(function() {
  script.targetMaterial.mainPass.baseColor = 
    new vec4(1, 0, 0, 1);
});

6Переходи між меню

Два Frame — головне меню і підменю з навігацією

Середньо

Що нового

  • Кілька Frame (MainMenu, SubMenu)
  • Навігація вперед/назад
  • Функції showMain() і showSub()

Код

// @input SceneObject mainMenu
// @input SceneObject subMenu
// @input Component.PinchButton openSubBtn
// @input Component.PinchButton backBtn

script.openSubBtn.onButtonPinched.add(function() {
  script.mainMenu.enabled = false;
  script.subMenu.enabled = true;
});

script.backBtn.onButtonPinched.add(function() {
  script.subMenu.enabled = false;
  script.mainMenu.enabled = true;
});

7Спавн + Видалення

Створюємо об'єкти і видаляємо їх кнопкою Clear

Середньо

Що нового

  • Масив spawnedObjects[] для зберігання посилань
  • Метод destroy() — видалити об'єкт
  • Очищення масиву після видалення

Код: Спавн

var spawnedObjects = [];

script.spawnBtn.onButtonPinched.add(function() {
  var obj = script.prefab.instantiate(script.container);
  spawnedObjects.push(obj);
  print("Створено. Всього: " + spawnedObjects.length);
});

Код: Видалення

script.clearBtn.onButtonPinched.add(function() {
  for (var i = 0; i < spawnedObjects.length; i++) {
    spawnedObjects[i].destroy();
  }
  spawnedObjects = [];
  print("Всі об'єкти видалено");
});

8Фільтри в меню

Кнопки-категорії показують різні типи об'єктів

Складно

Що нового

  • Масив prefab'ів categories[]
  • Змінна стану currentCategory
  • Функція showCategory(index)

Код

// @input Asset.ObjectPrefab[] categories
// @input Component.PinchButton[] categoryBtns

var currentCategory = 0;

function showCategory(index) {
  // Очистити контейнер
  // Заспавнити prefab з categories[index]
  currentCategory = index;
}

for (var i = 0; i < script.categoryBtns.length; i++) {
  (function(idx) {
    script.categoryBtns[idx].onButtonPinched.add(
      function() { showCategory(idx); }
    );
  })(i);
}

9Динамічні слайдери

Слайдери змінюють різні властивості: scale, rotation, opacity

Складно

Що нового

  • Кілька слайдерів для різних властивостей
  • Робота з Transform (position, rotation, scale)
  • Зміна прозорості матеріалу

Код: Scale

script.scaleSlider.onValueUpdate.add(function(value) {
  var scale = 0.3 + value * 2.7; // 0.3..3.0
  script.target.getTransform().setLocalScale(
    new vec3(scale, scale, scale)
  );
});

Код: Rotation

script.rotationSlider.onValueUpdate.add(function(value) {
  var angle = value * 360; // 0..360°
  script.target.getTransform().setLocalRotation(
    quat.fromEulerAngles(0, angle * Math.PI / 180, 0)
  );
});

Код: Opacity

script.opacitySlider.onValueUpdate.add(function(value) {
  var col = script.material.mainPass.baseColor;
  script.material.mainPass.baseColor = 
    new vec4(col.r, col.g, col.b, value);
});

🎮 Проєкт: Game Menu

Повноцінне ігрове меню з навігацією та функціоналом

Складно

Функції

  • Головне меню (Play, Settings, Exit)
  • Підменю Settings (Volume, Graphics)
  • Спавн ігрових об'єктів
  • Pause меню під час гри
  • Система станів (MainMenu, InGame, Paused)

Архітектура

MenuController
├─ currentState: string
├─ showMainMenu()
├─ startGame()
├─ pauseGame()
├─ resumeGame()
└─ exitGame()

UI Structure:
├─ MainMenuFrame
│  ├─ PlayButton
│  ├─ SettingsButton
│  └─ ExitButton
├─ SettingsFrame
│  ├─ VolumeSlider
│  └─ BackButton
└─ PauseFrame
   ├─ ResumeButton
   └─ QuitButton

Ключові концепції

  • State Machine — керування станами меню
  • Singleton — глобальний доступ через global.menuController
  • Callbacks — події для комунікації між скриптами
  • Lazy Loading — підвантаження підменю

Камера плавно слідує за об'єктом

Середньо

Що робить

  • Камера слідує за target об'єктом
  • Плавне переміщення через lerp()
  • Offset — зсув відносно target
  • Smooth factor — швидкість слідування

Код

// @input SceneObject target
// @input float smoothFactor = 0.1
// @input vec3 offset = {0, 5, -10}

function onUpdate() {
  var targetPos = script.target.getTransform()
    .getWorldPosition().add(script.offset);
  
  var currentPos = script.getTransform().getWorldPosition();
  
  var newPos = vec3.lerp(
    currentPos, 
    targetPos, 
    script.smoothFactor
  );
  
  script.getTransform().setWorldPosition(newPos);
}

var updateEvent = script.createEvent("UpdateEvent");
updateEvent.bind(onUpdate);
